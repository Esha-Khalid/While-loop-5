          
                                             While Loop Structure
Loop:
  A statement or set of statements that is executed repeatedly is known as loop.
      There are three types of loop.
  1. while loop
  2. do-while loop 
  3. for loop
 
   While(Condition)
   {
     Statement 1;
     Statement 2;
     Statement 3;
        .
        .
        .
     Statment N; 
    

   }
   
 NOTE:
    Program is executed when condition is True otherwise program is terminated.
